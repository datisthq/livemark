# Hello World

Welcome to the Livemark blog! This is our first post where we share updates about the project.

## What is Livemark?

Livemark is a markdown site generator that turns your `.md` files into a beautiful documentation website. It supports:

- GitHub Flavored Markdown
- Code highlighting with Shiki
- Math with KaTeX
- Mermaid diagrams
