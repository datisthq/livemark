# Authoring

Livemark accepts standard Markdown out of the box, plus a curated set of extensions for documentation sites — frontmatter, callouts, tabs, syntax-highlighted code, diagrams, math, and more. Each page below covers one area of the authoring surface.

## Features

:::card{title="Frontmatter" href="/authoring/frontmatter/" icon="file-text"}
YAML metadata for titles, descriptions, icons, and page ordering.
:::

:::card{title="Headings" href="/authoring/headings/" icon="heading"}
Auto-generated IDs, custom anchors, TOC control, and inline table of contents.
:::

:::card{title="Links" href="/authoring/links/" icon="link"}
Internal, external, anchor, and auto-detected links.
:::

:::card{title="Media" href="/authoring/media/" icon="image"}
Images with zoom, themed images, video and audio embeds.
:::

:::card{title="Text Formatting" href="/authoring/text-formatting/" icon="type"}
Bold, italic, lists, blockquotes, footnotes, definition lists, and emoji.
:::

:::card{title="Rich Elements" href="/authoring/rich-elements/" icon="layout-grid"}
Tables, callouts, tabs, steps, cards, badges, buttons, icons, and columns.
:::

:::card{title="Code Blocks" href="/authoring/code-blocks/" icon="code"}
Syntax highlighting, line numbers, diffs, annotations, and twoslash.
:::

:::card{title="Advanced Syntax" href="/authoring/advanced-syntax/" icon="puzzle"}
HTML blocks, MDX components, LaTeX math, Mermaid diagrams, and file includes.
:::
